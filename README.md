# ⚙ Installation

Clone repository:

```bash
git clone https://github.com/TalhaZeeshan-FullStackDeveloper/react-portfolio.git
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```

Build:

```bash
npm run build
```
